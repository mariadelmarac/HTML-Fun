# HTML-Fun

A personal portfolio website for Maria del Mar Acosta - Ingeniera Industrial. Built with HTML, CSS, optimized for SEO and accessibility.

## Project files
- [index.html](index.html) — main page (homepage)
- [pages/contact.html](pages/contact.html) — contact form page
- [pages/experience.html](pages/experience.html) — professional experience timeline
- [src/css/header.css](src/css/header.css) — header styles
- [src/css/main.css](src/css/main.css) — global layout styles
- [src/css/contact.css](src/css/contact.css) — contact form styles
- [src/css/experience.css](src/css/experience.css) — experience section styles
- [src/css/footer.css](src/css/footer.css) — footer styles
- [IMG/](IMG/) — image assets
- [robots.txt](robots.txt) — SEO configuration for search engines
- [sitemap.xml](sitemap.xml) — XML sitemap for search engines
- [.htaccess](.htaccess) — Apache server configuration (caching, compression)
- [README.md](README.md) — this file

## Features

### SEO Optimization
- ✅ Meta tags (description, keywords, author)
- ✅ Open Graph tags (og:title, og:description, og:image, og:url)
- ✅ Twitter Card tags for social media
- ✅ Canonical URLs on all pages
- ✅ Structured data (JSON-LD schema) for Person type
- ✅ Optimized page titles with keywords
- ✅ robots.txt and sitemap.xml
- ✅ .htaccess with caching and compression
- ✅ Proper language attribute (lang="es")

### Accessibility (ARIA)
- ✅ Semantic HTML5 elements
- ✅ ARIA labels and roles
- ✅ Alt text for images
- ✅ Form accessibility attributes

## Configuration Notes

### Update Domain URLs
Replace `mariaacosta.com` with your actual domain in:
- [index.html](index.html) — og:url, canonical link
- [pages/contact.html](pages/contact.html) — og:url, canonical link
- [pages/experience.html](pages/experience.html) — og:url, canonical link
- [sitemap.xml](sitemap.xml) — all URLs
- [robots.txt](robots.txt) — sitemap URL

### Add Favicon
Place a `favicon.ico` file in the [IMG/](IMG/) folder to display the site icon.

### Social Media Links
Update the social media URLs in the footer (Facebook, Instagram, LinkedIn) with your actual profiles.

### Contact Information
Update email and phone in:
- Footer sections of all pages
- Schema.org JSON-LD in index.html
